## Description

<p>A strobogrammatic number is a number that looks the same when rotated 180 degrees (looked at upside down).</p>

<p>Find all strobogrammatic numbers that are of length = n.</p>

<p><b>Example:</b></p>

<pre><b>Input:</b>  n = 2
<b>Output:</b> <code>["11","69","88","96"]</code>
</pre>
