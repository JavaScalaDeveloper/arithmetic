## Solutions

<!-- tabs:start -->

### **SQL**

```
SELECT (
		SELECT num
		FROM my_numbers
		GROUP BY num
		HAVING COUNT(*) = 1
		ORDER BY num DESC
		LIMIT 1
	) AS num
```
